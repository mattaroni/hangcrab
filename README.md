A rust-powered, CLI implementation of the classic game *Hangman*.

## Usage

> [!NOTE]
> Be sure to have [cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html)
> installed on your system.

```
cargo install --git https://github.com/mellowghostyx/hangcrab.git
```
